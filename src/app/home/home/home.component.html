Práctica final

Se mantiene el código que ha ido evolucionando a través de cada una de las prácticas semanales de modo que hay código redundante para activar partes de código en desuso en la práctica final, y finalmente, se deja el proyecto funcional interactuando mediante:
  • Contenedores-presentadores mediante @Inputs-@Outputs
  • Observables
  • Implementación de servicios
  • Consumiendo servicios remotos
  • Incorporando Angular Material a la tabla de proyectos
Por otra parte, se mantienen secciones "pre" para la visualización de datos, de modo que sea cómodo comprobar los datos guardados/enviados en determinados momentos.
